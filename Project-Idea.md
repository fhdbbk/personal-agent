# An agentic system working as a personal assistant

## Project Idea
I would like to build a personal agentic system that uses only open source small models which can be run on a laptop. It should be able to take instructions form either a chat window or an audio through microphone and then respond through speech or text after performing the actions asked. For UI we can use React. But I dont know much about React, so this will also be a learning for me. We can start with the goal of running it on a laptop CPU (16 GB RAM), but later we will also make for mobile either through an android app or through ReactNative. For speeding up different processes we can use C or C++ in combination with python.

Agent loop – the model can decide to use tools (code execution, file I/O, web search, calendar, etc.) rather than just answering. It should keep maintaining a long term memory of the user regarding his preferences and his behavioral pattern. It should also maintain a short term memory of the recent conversations
