# Session Logs

One file per working session: `YYYY-MM-DD-short-slug.md`.

Each entry should answer:
1. **Goal** — what we set out to do
2. **What we did** — the actual work, in order
3. **Decisions made** — link to ADRs in `../decisions/` if any were created
4. **Open threads** — what's left, what to pick up next
