# Learnings

Topic-based notes on what we discovered while building this. One file per topic — refactor/split when a topic grows too big.

Style guide:
- Lead with what you actually learned, not a tutorial.
- Link to the session(s) where you hit it: `[2026-05-01](../sessions/2026-05-01-phase-0-foundations.md)`.
- Capture the wrong turns too — the dead ends are often the most useful note for future-you.
